Intro

This website is a replication package for the paper “How Twitter divides the Dutch parliament: Segregation along party lines and social dimensions in the following, @-mentions and retweets Twitter Networks of Dutch MPs” by Tolsma and Spierings ((submitted)).

It contains R code to replicate all Tables/Figures/Appendix in the manuscript.

To copy the code click the button in the upper right corner of the code-chunks.

Use the top menu to navigate to the section of interest.

The source code of this website can be found on Github

Questions can be addressed to Jochem Tolsma.
